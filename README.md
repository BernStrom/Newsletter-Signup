## :wave: Intros!

This newsletter signup page is a simple full-stack web application that allow users to sign up for newsletter by inputting their name and email adddress. It is built with the Bootstrap 5 CSS framework along with some custom CSS styles on the client-side, and Express with Mailchimp's Marketing API to authenticate and add contacts to the marketing audience list on the server-side.

The live website is being deployed through Heroku and feel free to check it out [here](https://shielded-mesa-11553.herokuapp.com/ "Newsletter Signup")! As usual, any feedbacks on this project are always welcome. ðŸ™‰ ðŸ˜Š

## :pushpin: Objectives

* Create a newsletter signup page that notify users whether the subscription succeeded or failed.

* Utiltize a marketing API from third-party platform to store subscribed contacts in the marketing audience list.

* Authenticate whether the user is an existing or new contact.

* Deploy the web app using a PaaS such as Heroku, etc.
